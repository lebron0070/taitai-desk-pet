# 今天也没离职 v0.4.0 Beta

会站在当前软件窗口边缘、自己摸鱼的像素桌面宠物。

## 这一版有什么

- 在 Codex、浏览器、聊天、设计、文档和音乐软件之间自动跳转
- 优先站在当前窗口上沿；空间不足时会落在窗口内部
- 苔精灵、橘猫、柴犬三种形象
- 自动吃饭、睡觉、游泳、巡视、梳毛、追球和运动
- 软件切换时才移动，不跟随鼠标，也不会因为打字在窗口内乱跳
- 支持拖动、暂停、继续和右键互动

## 下载

Apple 芯片 Mac（M1 / M2 / M3 / M4）请下载：

`Taitai-0.4.0-arm64.dmg`

Windows 10/11 x64 请下载：

`Taitai-0.4.0-win-x64.exe`

不想安装的 Windows 用户可以下载：

`Taitai-0.4.0-win-x64.zip`

当前暂不支持 Intel Mac 和 32 位 Windows。
