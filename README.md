# 今天也没离职

一个会在不同应用之间跳来跳去、最后趴在当前窗口边缘的桌面上班搭子。支持 macOS 和 Windows，同时保留纯 H5 展示页。

## 本地运行

直接打开 `index.html`，或启动静态服务器：

```bash
python3 -m http.server 4173
```

然后访问 <http://localhost:4173>。

## 运行桌面宠物

```bash
npm install
npm run desktop
```

首次运行时，macOS 会询问“辅助功能”权限。允许后，宠物才能读取 Codex、浏览器等当前窗口的位置并趴在窗口边缘；未授权时会自动退化为趴在当前屏幕底部。Windows 使用系统前台窗口信息，不需要辅助功能权限。

## 构建 macOS 安装包

```bash
npm run build:mac
```

安装包将生成在 `release/`。未签名的本地测试版首次打开需要在“系统设置 → 隐私与安全性”中确认。

## 构建 Windows 安装包

请在 Windows 10/11 x64 或 GitHub Actions 的 Windows runner 上执行：

```bash
pnpm install
pnpm run build:win
```

将生成 `Taitai-0.4.1-win-x64.exe` 安装程序和 `Taitai-0.4.1-win-x64.zip` 免安装压缩包。推送 `v*` 标签时，`.github/workflows/release.yml` 会自动构建并附加到 GitHub Release。

## 当前公开测试系统

- macOS：Apple 芯片（M1 / M2 / M3 / M4），下载 `Taitai-0.4.1-arm64.dmg`
- Windows：Windows 10/11 x64，下载 `Taitai-0.4.1-win-x64.exe`
- 两个平台的 Beta 包暂未进行商业代码签名，只应从项目官方 GitHub Releases 页面下载

## 部署

整个目录可直接部署到 GitHub Pages、Vercel 或 Netlify。

## 当前功能

- 原创 CSS 工位宠物与待机动画
- 实时下班倒计时
- 投喂咖啡、摸头、一起叹气
- 本地保存功德、咖啡和互动数据
- 一键生成上班日报与复制分享文案
- `Esc` 老板键，快速切换到季度数据表
- 桌面与手机响应式布局
- Electron 透明、跨桌面、始终置顶窗口
- 识别当前前台应用和窗口边界
- Windows 10/11 前台窗口识别与窗口边缘停靠
- 切换到 Codex、浏览器等应用时跳到新窗口
- 只在当前前台软件发生变化时移动，不跟随鼠标
- 按软件类型选择窗口顶部或侧边作为停靠点
- 落地、观察、挥手和对话动作
- 鼠标悬停可交互，也可以拖动宠物位置
- 原创像素角色“苔苔”与终端脸表情
- 三种可选形象：苔精灵、橘猫、柴犬
- Codex、聊天、设计、浏览器、文档、音乐六类软件专属动作
- 9–18 秒随机待机行为，不持续打扰
- 本机设置：名称、性格、气泡、下班时间和开机启动
- 优先让角色双脚站在当前窗口上沿；屏幕顶部空间不足时落到窗口内部随机角落
- 软件切换不会因待机动作丢失，暂停后继续会强制重新识别当前窗口
- 本地养成状态：饱腹、清洁、心情、体力、知识和零花钱
- 右键“和它玩”：喂食、洗澡、运动、学习、打工、玩球和旅行
- 同一软件内部的面板和输入框变化不会触发重复降落
- 每隔约 3–7 秒自主选择动作，依据体力、饱腹和清洁状态自动休息、进食或游泳
- 放大版自主动作：趴睡、横向游泳、窗口巡视、梳毛、追球和大幅运动
